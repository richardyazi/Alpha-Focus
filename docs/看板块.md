# 看板块——识别"主力军团"

1. 涨幅榜前10板块
    - 是政策驱动（如"新质生产力"、"低空经济"）？
    - 是业绩催化（如年报预增）？
    - 是事件驱动（如AI大模型发布）？
2. 资金流向
    - 主力净流入最多的板块？连续3日流入？→ 主线可能形成
    - 跌幅前列板块是否集体出逃？→ 回避风险
3. 板块内部结构 
    - 龙头股：谁第一个涨停？封单强不强？（如常山北明之于华为算力）
    - 中军股：大市值代表是否稳住？（如宁德时代之于电池）
    - 跟风股：是否批量涨停？次日能否接力？
    
✅ 技巧：用“同花顺/东方财富”板块热度图快速定位活跃方向。
