# 代码修改总结

根据文档 `docs/少妇战法/聚宽选股说明.md` 的最新内容,对 `聚宽选股/key_kline_marker.py` 进行了以下修改:

## 主要修改

### 1. `batch_filter_stocks` 函数
**修改前**:
- 返回值: DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数', '涉及的板块'])
- 板块匹配支持精确匹配和关键字匹配(包含关键字)

**修改后**:
- 返回值: Tuple(DataFrame, dict)
  - 第一个元素: 符合B1的信息汇总 DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数', '涉及的板块'])
  - 第二个元素: 股票的B1标记数据 dict (key为股票代码,value为该股票的B1标记DataFrame)
- 板块匹配只支持精准匹配(删除了关键字匹配逻辑)
- 新增逻辑:在遍历每个板块时,同时获取并保存每只股票的B1标记数据

### 2. `filter_custom_stocks` 函数
**修改前**:
- 返回值: DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数', '涉及的自选组'])

**修改后**:
- 返回值: Tuple(DataFrame, dict)
  - 第一个元素: 符合B1的信息汇总 DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数', '涉及的自选组'])
  - 第二个元素: 股票的B1标记数据 dict (key为股票代码,value为该股票的B1标记DataFrame)
- 新增逻辑:在遍历所有股票时,同时保存每只股票的B1标记数据

### 3. `full_scan_b1_stocks` 函数
**修改前**:
- 返回值: DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数'])

**修改后**:
- 返回值: Tuple(DataFrame, dict)
  - 第一个元素: 符合B1的信息汇总 DataFrame (columns为['股票代码', '最近一次B1时间', 'B1出现次数'])
  - 第二个元素: 股票的B1标记数据 dict (key为股票代码,value为该股票的B1标记DataFrame)
- 新增逻辑:在遍历所有股票时,同时保存每只股票的B1标记数据

### 4. `calculate_concept_potential` 函数
**修改前**:
- 返回值: Tuple(DataFrame, DataFrame)
  - 概念板块统计: columns为['板块代码', '板块名称', '股票数量', '出现B1的股票数量', 'B1出现的总次数']
  -交易日统计: columns为['交易日期', 'B1出现的次数']

**修改后**:
- 返回值: Tuple(DataFrame, DataFrame)
  - 概念板块统计: columns为['板块代码', '板块名称', '出现B1的股票数量', 'B1出现的总次数'] (删除了'股票数量'列)
  - 交易日统计: columns为['交易日期', 'B1出现的次数'] (无变化)
- 新增逻辑:
  - 解包从其他函数获取的Tuple返回值
  - 优先使用已获取的B1标记数据,避免重复调用API
  - 统计逻辑中删除'股票数量'的统计

### 5. main函数中的示例代码
**修改**:
- 更新所有调用 `batch_filter_stocks`, `filter_custom_stocks`, `full_scan_b1_stocks` 的地方,使用Tuple解包接收返回值
- 示例:
  ```python
  # 修改前
  result = batch_filter_stocks(...)

  # 修改后
  result, b1_data_dict = batch_filter_stocks(...)
  ```

## 修改原因

根据文档要求:
1. **返回值格式变更**: 批量查询、自选股查询和全量扫描函数都需要返回Tuple(符合B1的信息汇总, 股票的B1标记数据)
2. **板块匹配方式变更**: 批量查询概念板块只支持精准匹配,不支持模糊匹配
3. **概念板块统计列名变更**: 删除了'股票数量'列,只保留'板块代码', '板块名称', '出现B1的股票数量', 'B1出现的总次数'

## 影响范围

- 所有调用上述三个函数的代码都需要相应调整
- `calculate_concept_potential`函数内部调用这些函数的地方已更新
- main函数中的示例代码已更新
