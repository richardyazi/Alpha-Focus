# 目的：
在同花顺远航版股票软件中实现特定K线的标记以及快捷选股；

# 关键K的判断规则：
参考文档 ./B1选股标准.md

# 输出区(粘贴AI生成内容):

同花顺公式实现方案
1. 创建技术指标公式
在同花顺软件中创建自定义指标公式：
```code
// B1选股指标公式
M1:=14;
M2:=28;
M3:=57;
M4:=114;

// 白线：EMA(EMA(C,10),10)
白线:EMA(EMA(C,10),10);

// 黄线：四重均线平均值
黄线:(MA(C,M1)+MA(C,M2)+MA(C,M3)+MA(C,M4))/4;

// 成交量条件
VOL_MA100:=MA(VOL,100);
量能条件:=VOL<VOL_MA100;

// 涨幅条件 (-2%~1.8%)
涨幅条件:=C/REF(C,1)>=0.98 AND C/REF(C,1)<=1.018;

// 振幅条件 (<7%)
振幅条件:=(H-L)/REF(C,1)<=0.07;

// KDJ J值条件
RSV:=(CLOSE-LLV(LOW,9))/(HHV(HIGH,9)-LLV(LOW,9))*100;
K:=SMA(RSV,3,1);
D:=SMA(K,3,1);
J:=3*K-2*D;
J值条件:=J<14;

// B1选股信号
B1信号:白线>黄线 AND C>黄线 AND 量能条件 AND 涨幅条件 AND 振幅条件 AND J值条件;

// 输出信号
// DRAWICON(B1信号,H*1.02,1);
DRAWTEXT(B1信号,H*1.04,'B1'),COLORFF75C6FF;
// STICKLINE(B1信号,H,L,0,0),COLORYELLOW;
// STICKLINE(B1信号,C,O,3,0),COLORYELLOW;
```
2. 使用方法
步骤1：创建公式

打开同花顺软件
按 Ctrl+F 打开公式管理器
选择"技术指标" → "新建"
将上述代码复制到公式编辑器中
命名为"B1选股信号"，保存
步骤2：应用到K线图

在个股K线图中，输入"B1选股信号"（或您设置的公式名称）
符合B1条件的K线会自动显示为黄色，并在K线上方标注"B1"
3. 公式功能说明
黄色K线：符合所有B1条件的K线会显示为黄色
B1文字标注：在符合条件的K线上方显示"B1"文字
图标标记：在K线最高点上方显示标记图标

4. 选股功能（可选）
如果您还需要选股功能，可以创建选股公式：
```code
// B1选股公式（用于条件选股）
M1:=14;M2:=28;M3:=57;M4:=114;
白线:=EMA(EMA(C,10),10);
黄线:=(MA(C,M1)+MA(C,M2)+MA(C,M3)+MA(C,M4))/4;
VOL_MA100:=MA(VOL,100);

RSV:=(CLOSE-LLV(LOW,9))/(HHV(HIGH,9)-LLV(LOW,9))*100;
K:=SMA(RSV,3,1);
D:=SMA(K,3,1);
J:=3*K-2*D;

B1选股:白线>黄线 AND C>黄线 AND VOL<VOL_MA100 AND 
       C/REF(C,1)>=0.98 AND C/REF(C,1)<=1.018 AND 
       (H-L)/REF(C,1)<=0.07 AND J<14;
```
5. 注意事项
参数修正：您文档中第6行有误，应为 M1:14, M2:28, M3:57, M4:114，已修正
创业板/ST过滤：同花顺选股时可以在条件设置中直接过滤创业板和ST股
实时更新：公式会实时计算，新数据符合条件时会自动标注
