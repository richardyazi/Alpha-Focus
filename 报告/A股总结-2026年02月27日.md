## 3. 综合研判与操作建议

### 3.1 市场核心特征
1. **主线明确**：**资源股（有色金属、电力）** 成为绝对主线，受涨价+政策双重驱动
2. **资金调仓**：资金从高位科技股流向低位资源股，风格切换明显
3. **指数分化**：上证强势，创业板弱势，反映市场风险偏好变化
4. **量能健康**：虽略有缩量，但2.5万亿成交额支撑结构性行情

### 3.2 风险提示
1. **技术压力**：上证面临前高4180-4200点压力，需放量突破
2. **创业板风险**：跌破5日线，技术形态转弱，需谨慎
3. **板块轮动**：资金快速切换，追高风险增加
4. **业绩窗口**：2月收官，3月进入年报密集披露期，警惕业绩雷

### 3.3 操作建议
1. **紧跟主线**：聚焦有色金属（钨、锑、稀土）、电力板块龙头
2. **控制仓位**：指数分化加剧，建议仓位控制在5-7成
3. **避高就低**：回避高位科技股（电子、通信），关注低位资源股
4. **关注量能**：上证突破前高需放量至2.8万亿以上，否则考虑高抛
5. **重点标的**：
   - 有色金属：章源钨业、北方稀土、包钢股份
   - 电力：珈伟新能、涪陵电力、南网能源
   - 算力租赁：拓维信息、云天励飞（短线博弈）
